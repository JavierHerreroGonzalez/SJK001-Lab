# SJK001-Lab
## Overview
This repository contains the implementations of the controllers for the Visual Follow Line and Drone Rescue People exercises in Unibotics Robotics Academy. This repository was created for the UJI Master in Intelligent Systems subject SJK001-Cyber-physical and Robotic Intelligent Systems.

## Structure
Inside each folder are the controller implementations along with a README file detailing the respective exercise and explaining the code.
